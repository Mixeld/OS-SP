dirwalk — это утилита командной строки для рекурсивного обхода файловой системы, 
аналогичная команде find. Она выводит пути к файлам, каталогам и символическим ссылкам, 
с возможностью фильтрации по типу и сортировки вывода. Программа написана на языке C 
с использованием только стандартных функций POSIX, 
что обеспечивает её переносимость между различными Unix-подобными операционными системами.
